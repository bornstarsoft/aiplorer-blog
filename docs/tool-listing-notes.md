# Tool Listing Notes

Date: 2026-06-01

`/ai-tools/tools/` is the reviewed AI tool index for Aiplorer.

## Listing Rules

- Normal production builds should show only published tool pages with
  `reviewStatus: "reviewed"`.
- Draft pages are allowed for workflow validation, but they should not be linked
  publicly from the reviewed tool index.
- Tool pages should be published in small manually reviewed groups.
- The listing should remain simple until there is enough reviewed content to
  justify filters, search, or comparison views.

## Current Reviewed Tools

- ChatGPT
- Claude
- Gemini

## Future Additions

Future batch additions should continue using cautious wording, official-source
review, no fake ratings, no fake rankings, and no unsupported "best" claims.
