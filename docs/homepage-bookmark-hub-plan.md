# Aiplorer Homepage Bookmark Hub Plan

Date: 2026-06-05

The Aiplorer homepage is positioned as a lightweight AI tools bookmark hub. It
should help everyday users quickly reach reviewed tool pages, practical
categories, guides, and use cases without turning the site into a thin link
directory.

## Intent

- Keep the homepage fast, simple, and static.
- Highlight reviewed tools that already passed the manual review workflow.
- Preserve the main service structure: AI Tools, Guides, Use Cases, and Blog.
- Make common entry points easy to scan for returning users.
- Avoid rankings, ratings, affiliate links, ads, search, automation, and
  unsupported claims.

## Homepage Structure

- Start with Aiplorer's positioning as a quick AI tools bookmark hub.
- Link to the reviewed AI tools index.
- Provide quick bookmarks for the reviewed tool pages.
- Provide category links for writing, image, video, audio, coding,
  productivity, automation, learning, and business tools.
- Provide the first guide and use-case links.
- Include cautious wording that features, pricing, and availability may change.

## Quality Bar

The homepage should remain a reviewed-tool hub, not a raw directory of every AI
product. New tool links should appear on the homepage only after the tool page
has been reviewed, published, and verified in production.
